# 🤳 ScrollSet
### **Watch what matters. Learn from your scroll**
ScrollSet is a React Native mobile application that helps users take back control of their screen time without giving up short-form content.

## 📖 Purpose
Instead of relying on recommendation algorithms, ScrollSet lets users decide exactly what they want to watch by filtering content around self-selected topics. The goal is to make every scroll intentional, educational, and aligned with the user's interests.

## ✨Features
- 🎯 Topic-based content filtering
- 🚫 No recommendation algorithm
- ⏱️ Custom scrolling session limits
- 🧠 Intentional learning-focused experience
- 📱 Clean and minimal mobile interface

## 🛠️ Tech Stack
- React Native
- Expo
- TypeScript
